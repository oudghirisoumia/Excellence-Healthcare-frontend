
# Excellence Parapharmacie - React (Complète)

Projet React simple reproduisant la page d'accueil de la maquette.

## Installer et lancer

1. Extraire l'archive.
2. Dans le dossier du projet :
```bash
npm install
npm start
```
3. Ouvrir http://localhost:3000

## Contenu
- React + react-router-dom
- Pages: Accueil, Catalogue
- Composants: Header, Categories, Hero, Footer
- Style simple en CSS dans `src/index.css`

Tu peux personnaliser les images, textes et ajouter l'intégration d'une API plus tard.
